== Description ==

Display your Domains Webwiki rating. The Webwiki plugin allows users to see how trustworthy your website is and gives them the opportunity to leave a feedback on Webwiki. 

Webwiki is a free domain directory which acquires technical and content information about a website in order to allow users getting a neutral first impression before visiting the website. All comments left on Webwiki are pre moderated so your domain entry will not be spammed with comments  or receive unfair criticism.

The Webwiki rating button can also be generated purely as HTML without using this plugin. Just use the following link: http://www.webwiki.com/info/webwiki-buttons.html

== Installation ==

1. Upload the webwiki-siterateing folder (unzipped) to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Design > Widgets and drag the Webwiki Widget to your sidebar
1. Configure the widget and hit save
